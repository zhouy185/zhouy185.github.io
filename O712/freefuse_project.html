📌FreeFuse Project – Interactive Video Pathways

🏢Business Context

FreeFuse produces interactive, “choose‑your‑own‑adventure” videos where viewers follow branching paths. The company’s proprietary engagement metric, Path IQ, aims to link viewing behavior to real outcomes (retention, conversions, learning). Watch this 42 second video for the idea.

🧾The Data

Anonymized user IDs Node/segment titles Timestamps Watch duration Branch choices Questionnaire flags

These fields let you reconstruct journeys, measure completions/drop‑offs, and analyze decision patterns.

🎯Project Goals (via data analysis)

  1. Validate Path IQ → Does it correlate with engagement & positive outcomes?
  2. Quantify impact → How large is the lift/ROI from Path IQ & pathway features?
  3. Guide resourcing → Should content/resources follow observed viewing behavior?
